# Fruit Inspection

## Overview
The food inspection project involves the building of models to locate the imperfections and defects on fruits. The fruits contain defects that can be located using a combination of edge detection and colour segmentation. More information on the project tasks can be found [here](https://github.com/LeonidasY/master-ai-projects/blob/main/fruit-inspection/project-tasks.pdf).

### Dataset
The dataset consists of 3 folders for each task:

- first task : 3 pairs of images of apples with defects. Each pair consists of a grey scale and a RGB image.
- second task : 2 pairs of images of apples with defects. Each pair consists of a grey scale and a RGB image and each apple is of a different colour.
- final challenge : 5 pairs of images of kiwis with and without defects. Each pair consists of a grey scale and a RGB image.
