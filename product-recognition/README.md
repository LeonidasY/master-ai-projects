# Product Recognition

## Overview
The product recognition project involves the building of a model to recognition and outline boxes of cereals of different brands from store shelves. More information on the project tasks can be found [here](https://github.com/LeonidasY/master-ai-projects/blob/main/product-recognition/project-tasks.pdf).

### Dataset
The dataset consists of 2 folders for all 3 tasks:

- models : 27 images of cereals of various brands and product names.
- scenes : 15 images of store shelves with various cereal boxes.

## Results
The project is included as a .ipynb file and can be viewed using the [Jupyter nbviewer link](https://nbviewer.jupyter.org/github/LeonidasY/master-ai-projects/blob/main/product-recognition/product-recognition.ipynb) or via Jupyter notebook directly on a desktop.
